# 📅 Daily Habit Tracker

> **Build better habits. Stay consistent. Achieve your goals.**

A lightweight, responsive web app designed to help you effortlessly track your daily routines and build momentum with a clean, distraction-free interface.

## ✨ Features

- ✅ **Daily Tracking** — Seamlessly add, update, and manage your daily habits
- ⚡ **Lightning Fast** — Built with Vite for instant loading and high performance
- 🎨 **Minimalist UI** — Clean, intuitive, and standard CSS design to keep you focused
- 📱 **Responsive** — Works perfectly on mobile, tablet, and desktop
- 💾 **No Login Required** — Easy to use right out of the box, data saves to local storage

## ⚙️ Getting Started

### Prerequisites

  - Node.js v18+
  - npm v9+

### Installation

```bash
# Clone the repository
git clone [https://github.com/shivanshgupta01/habit-tracker.git](https://github.com/shivanshgupta01/habit-tracker.git)

# Navigate into the project
cd habit-tracker

# Install dependencies
npm install
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:5173](https://www.google.com/search?q=http://localhost:5173) ✅

## 🚀 Deployment on Vercel

### Step 1 — Push to GitHub

```bash
git add .
git commit -m "initial commit"
git push origin main
```

### Step 2 — Deploy on Vercel

1.  Go to **vercel.com** and import your repository
2.  Click **Deploy** ✅

### Step 3 — Future Updates

Every time you make changes just run:

```bash
git add .
git commit -m "your update message"
git push origin main
```

Vercel auto-deploys in 30 seconds ✅
